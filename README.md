Geo Snap Viewer 360
Geo Snap Viewer 360 は、写真に含まれる GPS 情報をもとに、
地図上に撮影地点を可視化する Web アプリケーションです。
🔧 機能
- 写真の GPS 情報を解析して地図に自動プロット
- 360°写真の自動判定とパノラマビュー表示
- サイドバーで写真一覧を管理
- PC のドラッグ＆ドロップ対応
- スマホ最適化 UI
- 初回のみサンプル写真を表示
📦 使用ライブラリ
- Leaflet（BSD-2-Clause）
- Leaflet MiniMap（MIT）
- Pannellum（MIT）
- EXIF.js（MIT）
